RVL UR-Robotiq Integrated DriverÔÉÅ

The Robotic Vision Laboratory (RVL) UR-Robotiq Integrated Driver is a wrapper for existing ROS services and topics provided by the Universal Robots (UR) ROS Driver. It has additional support for a Robotiq gripper (e.g., 2F) directly attached to the UR tool port. The driver also provides functions for actuating the gripper in various modes (e.g., grasp on open/close, specific jaw gap, etc.) as well as integrating MoveIt! motion planning for tracking the entire setup, UR robot, and Robotiq gripper.

At the time of this release, the driver is working on our setup with both Windows 10 (Version 10.0.19044 Build 19044) and Ubuntu 20.04.
